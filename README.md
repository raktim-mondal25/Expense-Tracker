# Java Expense Tracker (JDBC + MySQL)

A console-based Expense Tracker application built using Java, JDBC, and MySQL.

## Features
- Add new expenses
- View all expenses
- Filter expenses by category
- Monthly total calculation
- Delete an expense
- MySQL database integration

## Tech Stack
- Java (OOP + Collections)
- JDBC
- MySQL

## How to Run
1. Import project into IntelliJ/Eclipse
2. Run SQL script to create database
3. Update MySQL username/password in DBConnection.java
4. Run Main.java
